# Weather App

A simple weather app built with HTML, CSS, and JavaScript. It fetches live weather data from the OpenWeather API based on a city name entered by the user.

## Live Demo

https://weather-app-aiz1.onrender.com

## Features

- Search weather by city name
- Displays temperature, condition, humidity, wind speed, and "feels like" temperature
- Shows a weather icon based on current conditions
- Error handling for invalid city names

## Tech Stack

- HTML
- CSS
- JavaScript (fetch API, async/await)
- [OpenWeather API](https://openweathermap.org/api)

## How to Run Locally

1. Clone this repository
2. Open `index.html` in your browser
3. Enter a city name and click Search
